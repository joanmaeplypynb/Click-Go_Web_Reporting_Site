KEY ISSUES IDENTIFIED

Some parts of the system still need UI/UX improvement, as users found certain areas less intuitive and visually inconsistent.

Clear instructions or guidance are limited in some parts of the system, which may cause confusion for users during tasks.

The visual design of the system can still be improved to make the interface more appealing and consistent.

Loading speed in some parts of the system may affect user experience, making the process less smooth and responsive.

DISCUSSION OF FINDINGS

The results of the usability evaluation show that the Click&Go prototype was generally well received by users, with most participants finding the system easy to use, easy to learn, and suitable for frequent use. Users also reported feeling confident while navigating the platform, which indicates that the interface provides enough clarity and guidance. In addition, many respondents did not strongly feel the need for technical support, suggesting that the system is understandable even for first-time users. However, some minor usability issues were still identified, particularly in navigation, button consistency, and error prevention. Some users experienced slight complexity in certain parts of the booking process, while others suggested clearer confirmation prompts and more visible instructions before completing important actions. These findings show that while the prototype performs well in overall usability, improving consistency and strengthening confirmation features can further enhance the user experience.

RECOMMENDATIONS FOR IMPROVING THE PROTOTYPE

Improve the overall UI/UX by making the system more intuitive, user-friendly, and visually consistent for all users.

Add clearer instructions and guidance within the system to help users complete tasks smoothly and reduce confusion.

Enhance the visual design by improving layout consistency and overall appearance to make the interface more appealing.

Optimize loading speed to improve system responsiveness and provide a smoother user experience during booking and navigation.

SUGGESTED DESIGN IMPROVEMENTS

Improve UI/UX

Make improvements to the overall usability to ensure the interface is easier to use and more consistent across all screens.

Add Simple Instructions

Provide clear and concise guidance in some parts of the system to help users understand what to do.

Improve Visual Design

Enhance the layout and styling to make the interface more organized and visually consistent.

Improve Performance

Optimize loading speed in certain parts of the system to ensure smoother user interaction.
