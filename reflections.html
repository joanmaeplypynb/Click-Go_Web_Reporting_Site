The Click&Go Project Journey

CLICK&GO PROJECT JOURNEY

The Click&Go project was created not only for academic requirements but also to understand UX and system design better. Our main goal was to develop a system that helps users book and travel in an easier and faster way. Through this project, we learned that design is not only about appearance but also about how users interact with the system.

We used a user-centered design approach during development. With the help of the Heuristic Evaluation Workbook, we were able to identify usability issues and improve the system. We focused on making the navigation simple, buttons clear, and layout consistent so users would not get confused while using the system.

Teamwork played a big role in completing this project. We conducted data gathering using Google Forms and Google Meet. Each member had specific tasks such as facilitating, recording responses, and observing user behavior during testing. This helped us organize our work and complete the project efficiently.

The video demonstration showed the final output of our system. It helped us see how users interact with Click&Go and allowed us to identify areas that still need improvement.

In conclusion, this project gave us valuable experience and knowledge that we can use in future studies and careers in technology and system design.
